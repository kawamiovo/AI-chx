# 💘 AI 虚拟前女友 (Context-Aware AI Girlfriend)

> 一个拥有**长期记忆**、**情感状态机**和**性格模仿能力**的 AI 虚拟伴侣项目。
> 她不仅记得你们很久以前聊过的话题，还会根据你的态度开心或生气，拒绝机械式的 AI 回复。

---

## 📂 1. 项目文件大揭秘 (小白必看)

在这个项目中，每个文件都扮演着身体的一个器官。以下是详细说明：

### 🧠 核心大脑
*   **`main_agent.py` (总指挥)**：
    *   这是 AI 的**大脑皮层**。它负责接收你的消息，决定该调用哪块记忆，判断当前的心情（开心还是生气），组装 Prompt（提示词），最后指挥 LLM 说话。
    *   **核心功能**：情感计算、用户画像更新 (`user_info`)、回复生成。

*   **`app.py` (嘴巴和耳朵)**：
    *   这是一个 Web 服务器（基于 Flask）。它负责把网页界面（前端）和你展示给浏览器，接收你在网页上输入的话，传给大脑，再把大脑的回复传回网页。

*   **`config.py` (身份证与钥匙)**：
    *   存放配置信息的地方。包括 API Key（你的账号密码）、模型选择、数据库维度设置等。**不要把这个文件发给陌生人**。

### 💾 记忆系统 (海马体)
*   **`memory_core.py` (深层潜意识)**：
    *   **最牛的文件**。它负责读取 `ex_gf.csv`（聊天记录），将成千上万条记录转化为向量（数学数字），存入向量数据库。
    *   当你问“你在哪”时，它能瞬间找出你们以前聊过类似的场景，让 AI 模仿当时的语气。

*   **`conversation_history.py` (短期记忆)**：
    *   负责记住**最近 20 句**对话，保证聊天连贯性（比如你上一句说“吃饭”，下一句说“好撑”，它知道你是吃饭撑的）。同时负责把新的对话异步存入硬盘。

*   **`conversation.py` (连接器)**：
    *   一个适配器小工具。它像个翻译官，帮助数据库理解我们的 Embedding 服务。

*   **`conversation_2.py` (旧大脑)**：
    *   *已废弃/备份*。这是上一代的关键词匹配逻辑，现在的项目主要运行在 `memory_core.py` 之上，这个文件仅作参考或备份。

### 🔧 工具与驱动
*   **`llm.py`**: 负责和远程的大模型（如 DeepSeek, GPT）打电话的工具。
*   **`embedding.py`**: 负责把文字变成向量（Embedding）的工具。它处理了网络重试、超时等复杂问题。
*   **`requirements.txt`**: 购物清单。列出了运行这个项目需要安装的所有 Python 库。

### 📁 数据文件
*   **`ex_gf.csv`**: **原料**。你导出的微信聊天记录。
*   **`save/` 目录**:
    *   `me.txt`: AI 记住的关于你的事实（动态更新）。
    *   `mood.txt`: AI 当前的心情分数。
    *   `memory_advanced/`: 存放长期语气记忆的数据库（机器看的文件）。
    *   `chat_history/`: 存放所有历史对话的数据库。
    *   `log/`: 给人看的流水账日记。

## 🛠️ 2. 快速上手指南

### 第一步：环境准备
1.  确保你安装了 **Python 3.8** 或以上版本。
2.  打开终端（CMD 或 PowerShell），进入项目文件夹。
3.  安装依赖库（照着购物清单买东西）：
    ```bash
    pip install -r requirements.txt
    ```

### 第二步：配置账号 (`config.py`)
打开 `config.py`，填入你的 API 信息：
```python
class Config:
    # 你的大模型 API 地址和 Key
    LLM_API_URL = "https://twohornedcarp.com/v1" 
    LLM_API_KEY = "sk-你的Key"
    
    # 你的向量模型配置 (必须与 Embedding API 对应)
    EMBEDDING_API_KEY = "sk-你的Key"
    EMBEDDING_DIMENSION = 1024  # 注意：必须确认你的模型维度是 1024 还是 4096！
```

### 第三步：注入记忆 (仅第一次需要)
**这一步是为了让 AI 拥有“前女友”的灵魂（语气和过往记忆）。**

1.  准备好你的聊天记录 CSV 文件，确保里面有 `StrContent` (内容) 和 `IsSender` (是谁发的，1是你，0是她) 这两列。
2.  打开 `memory_core.py`。
3.  找到底部的 `main` 函数，**解开这行代码的注释**：
    ```python
    # await memory_system.ingest_excel(excel_path)  <-- 删掉前面的 # 号
    ```
4.  运行这个文件进行记忆注入：
    ```bash
    python memory_core.py
    ```
    *等待进度条走完，看到“🎉 记忆注入完成”字样。*
5.  **重要**：注入完成后，记得**重新把那行代码注释掉**（加上 `#`），否则下次运行又会重复注入。

### 第四步：启动虚拟女友
运行主程序：
```bash
python app.py
```
看到 `Running on http://127.0.0.1:5000` 后，打开浏览器访问这个地址，即可开始聊天！

## ❓ 3. 常见问题 (小白救星)

**Q1: 报错 `Dimension 1024 does not match collection dimensionality 768` 怎么办？**
*   **原因**：你换了更高级的手机壳（1024维），但还在用旧手机（768维的数据库）。
*   **解决**：
    1. 关闭程序。
    2. 删除 `save/` 文件夹下的 `chat_history` 和 `memory_advanced` 文件夹。
    3. 重新运行程序，它会自动创建新的。

**Q2: 我想重置她的记忆，让她忘掉我是谁？**
*   **解决**：删除 `save/me.txt` 文件和 `save/chat_history` 文件夹。

**Q3: 她说话太冲了/太舔狗了，怎么调整？**
*   **解决**：打开 `save/mood.txt`。
    *   改成 `10`：她会变成恶龙，极度暴躁。
    *   改成 `90`：她会变成甜妹，超级粘人。
    *   或者直接在聊天里哄她/骂她，她的心情会自动变化。

**Q4: 报错 `Event loop is closed`？**
*   **原因**：程序退出的姿势不对，或者后台任务冲突。
*   **解决**：确保你使用的是最新版的 `conversation_history.py`（使用了 `threading` 那个版本）。

## 🌟 核心特性说明

1.  **情感状态机**：
    她不是固定的性格。你回复得敷衍，她心情值下降，回复就变短、变冷；你哄她，她心情值上升，回复变甜、变长。

2.  **RAG 语气模仿**：
    当你说“在干嘛”时，她不会回复通用的“在发呆”，而是会去搜索你们几年前聊天记录里她回复“在干嘛”的那句话（比如“关你屁事”），从而实现惊人的逼真度。

3.  **增量记忆**：
    你说“我下周去北京”，她会悄悄记在小本本（`me.txt`）上。下周你问她“我去哪”，她能答上来。
